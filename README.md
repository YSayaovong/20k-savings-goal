# 20K Savings Dashboard (VS Code)

Senior-level data analyst project that models a bi-weekly savings plan with monthly compounding, produces a cashflow table, exposes SQL KPIs, and ships an interactive Streamlit dashboard.

## Defaults
- Starting balance: **$143.24**
- Deposits: **$100 every 2 weeks** (26 per year)
- APY: **3.8%**, compounded monthly
- Goal: **$20,000**

## Run (VS Code or Terminal)
```bash
# from project root
conda create -n savings python=3.11 -y  # optional (Anaconda)
conda activate savings                  # optional
pip install -r requirements.txt

python pipeline/generate_cashflows.py
python pipeline/load_to_sqlite.py

streamlit run app/app.py
```

## Business Need
Turn a vague savings goal into a precise, trackable plan with clear KPIs: months to target, contribution vs. interest, run-rate, and scenario analysis. Useful for personal finance coaching or fintech feature prototyping.

## Notes
- Everything runs locally (no PostgreSQL required). SQLite DB is stored at `data/outputs/savings.db`.
- Use DuckDB in notebooks if preferred (`duckdb:///savings.db`).
